# 🐐 AL-WARD Farm Analytics

> Data-driven analysis of livestock productivity,
> nutritional efficiency, and financial ROI.

## 📊 Project Overview
Quantitative analysis of the AL-WARD Beetal Farm 
operational data. This project identifies high-margin 
bloodlines and optimizes feed-to-weight conversion 
ratios using SQL Window Functions and predictive modeling.

## 🛠️ Tech Stack
- **Languages:** SQL (CTEs, Window Functions), Python (Pandas, Scikit-Learn)
- **Visualization:** Tableau Public, Streamlit
- **Infrastructure:** Git (Conventional Commits), DuckDB

## 🎯 Primary Objectives
- **Financial ROI:** Identifying top-performing bloodlines based on purchase-to-sale margins.
- **Expense Optimization:** Analyzing feed consumption patterns to reduce waste and maximize weight gain.
- **Predictive Modeling:** Estimating future livestock value based on age, weight, and pedigree.
